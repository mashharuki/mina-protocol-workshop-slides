# workshop-slides

Code to go with the o1js workshop slides(DEVCON7)

## Steps

1. Create program: 2024-devcon-create-program
2. Write tests: 2024-devcon-write-tests
3. Incorporate UI: 2024-devcon-ui

## 動かし方(ZK 回路側)

- インストール

  ```bash
  cd zk && bun install
  ```

- テスト

  ```bash
  bun run test
  ```

- ビルド

  ```bash
  bun run build
  ```

## 動かし方(フロントエンド)

- インストール

  ```bash
  cd ui && bun install
  ```

- 起動方法

  ```bash
  bun run dev
  ```
